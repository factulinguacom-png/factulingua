Product name: FactuLingua Desktop
Vendor: FactuLingua
Version: 1.0.2

1. Overview
FactuLingua Desktop is a professional application for managing invoices,
quotes and related business documents across multiple languages and
jurisdictions. The application is designed for small and medium-sized
businesses that require reliable document workflows, secure data handling
and integration with a cloud-based backend.

2. System Requirements
 - Operating system: Windows 10 or later (64-bit)
 - Memory: 4 GB RAM (8 GB recommended)
 - Disk space: 1 GB free space for application and updates
 - Network: Stable internet connection for authentication, syncing and updates
 - Permissions: Ability to install software in the selected directory

3. Installation
The FactuLingua Desktop installer will:
 - Copy application files to the selected installation directory
   (default: C:\Program Files\FactuLingua\).
 - Create a Start Menu entry and an optional desktop shortcut.
 - Register the installation path in the system for future updates
   and support diagnostics.
During setup, you will be asked to accept the End User License Agreement
(EULA) and review the Privacy Policy / Data Processing terms.

4. First Launch and Sign-In
On first launch, you will be asked to:
 - Sign in using your existing FactuLingua account, or
 - Create a new account using your email address or supported SSO provider.
An active internet connection is required to authenticate and to
synchronise your subscription and entitlements.

5. Updates
FactuLingua Desktop may periodically check for updates using the official
update service. When a new version is available, you may be prompted to
download and install the update. In managed environments, updates may be
distributed centrally by your IT department.

6. Data Storage and Privacy
FactuLingua Desktop stores:
 - Application binaries in the installation directory,
 - User-specific configuration and cache in the application data directory,
 - Business data and documents primarily in the secure server-side backend.
For details on data processing, retention and security, please refer to
PrivacyPolicy.rtf and the Data Processing Agreement.
